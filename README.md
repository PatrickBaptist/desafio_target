Este repositório contém desafios resolvidos em C#:

1. Cálculo de comissão a partir de um JSON.
2. Movimentação de estoque com validações.
3. Cálculo de juros diários (2,5% ao dia).

## Pré-requisitos

- .NET SDK 7.0 ou superior
- Sistema operacional: Windows, Linux ou MacOS

## Como rodar cada desafio

Entre na pasta do desafio e execute:

dotnet run

## Como rodar todos de uma vez

### Windows:
./run_all.ps1

### Linux/Mac:
./run_all.sh
